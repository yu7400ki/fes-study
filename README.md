## Local Development

本書は[HonoX](https://github.com/honojs/honox)で作成されています。
また、パッケージマネージャーには[Bun](https://github.com/oven-sh/bun)を使用しています。

```bash
bun i
bun run dev
```

起動後、`http://localhost:5173`にアクセスすることで、ローカルでの開発が可能です。
