# セレクターの基本

セレクターは、スタイルを適用する要素を選択するための記法です。

## 目次

## 構文

```css
selector {
  property: value;
}
```

## 要素型セレクター

要素型セレクターは、要素の名前で要素を選択します。

<WebExample>
  <style>
    {`
      h1 {
        color: red;
      }
    `}
  </style>
  <h1>Hello</h1>
  <h2>World</h2>
</WebExample>

```css
h1 {
  color: red;
}
```

```html
<h1>Hello</h1>
<h2>World</h2>
```

上記の例では、`h1`要素に対して`color`プロパティに`red`を選択しています。

## クラスセレクター

クラスセレクターは、ピリオド（`.`）を先頭に付けてクラス名で要素を選択します。

<WebExample>
  <style>
    {`
      .red {
        color: red;
      }
    `}
  </style>
  <p class="red">Hello</p>
  <p>World</p>
</WebExample>

```css
.red {
  color: red;
}
```

```html
<p class="red">Hello</p>
<p>World</p>
```

## IDセレクター

IDセレクターは、ハッシュ（`#`）を先頭に付けてID名で要素を選択します。

<WebExample>
  <style>
    {`
      #red {
        color: red;
      }
    `}
  </style>
  <p id="red">Hello</p>
  <p>World</p>
</WebExample>

```css
#red {
  color: red;
}
```

```html
<p id="red">Hello</p>
<p>World</p>
```

## 全称セレクター

全ての要素を選択するセレクターです。
全称セレクターは、アスタリスク（`*`）で選択します。

<WebExample>
  <style>
    {`
      * {
        color: red;
      }
    `}
  </style>
  <p>Hello</p>
  <p>World</p>
</WebExample>

```css
* {
  color: red;
}
```

```html
<p>Hello</p>
<p>World</p>
```

## 属性セレクター

属性セレクターは、角括弧（`[]`）で囲み、属性や属性の値で要素を選択します。

| セレクター | 説明 |
| --- | --- |
| `[attr]` | `attr`属性を持つ要素 |
| `[attr="value"]` | `attr`属性で値が`value`と完全一致する要素 |
| `[attr~="value"]` | `attr`属性で値が`value`を含む要素 |
| `[attr\|="value"]` | `attr`属性で値が`value`または`value-`で始まる要素 |
| `[attr^="value"]` | `attr`属性で値が`value`で始まる要素 |
| `[attr$="value"]` | `attr`属性で値が`value`で終わる要素 |
| `[attr*="value"]` | `attr`属性で値が`value`と部分一致する要素 |

<WebExample>
  <style>
    {`
      [class] {
        font-size: 20px;
      }
      [class=red] {
        color: red;
      }
      [class~=bold] {
        font-weight: bold;
      }
      [class|=strike] {
        text-decoration: line-through;
      }
    `}
  </style>
  <ul>
    <li>First</li>
    <li class="red">Second</li>
    <li class="red bold">Third</li>
    <li class="strike-through">Fourth</li>
  </ul>
</WebExample>

```css
[class] {
  font-size: 20px;
}

[class="red"] {
  color: red;
}

[class~="bold"] {
  font-weight: bold;
}

[class|="strike"] {
  text-decoration: line-through;
}
```

```html
<ul>
  <li>First</li>
  <li class="red">Second</li>
  <li class="red bold">Third</li>
  <li class="strike-through">Fourth</li>
</ul>
```

- `[class]`は、`class`属性を持つ全ての要素に対して`font-size`プロパティに`20px`を選択しています。
- `[class="red"]`は、`class`属性で値が`red`と完全一致する要素に対して`color`プロパティに`red`を選択しています。`red`クラスを持つ`Third`には適用されていないことに注意してください。
- `[class~="bold"]`は、`class`属性で値が`bold`を含む要素に対して`font-weight`プロパティに`bold`を選択しています。
- `[class|="strike"]`は、`class`属性で値が`strike`または`strike-`で始まる要素に対して`text-decoration`プロパティに`line-through`を選択しています。

本書のサイドバーにて、閲覧中のページのリンクの色が変わっていることに気づいた方もいるでしょう。これは、`aria-current="page"`属性を持つリンクに対してスタイルを適用しているためです。

## セレクターリスト

カンマ（`,`）でセレクターを区切ることで複数のセレクターを指定することができます。

<WebExample>
  <style>
    {`
      h1, h2 {
        color: red;
      }
    `}
  </style>
  <h1>Hello</h1>
  <h2>World</h2>
</WebExample>

```css
h1, h2 {
  color: red;
}
```

```html
<h1>Hello</h1>
<h2>World</h2>
```

## セレクターを組み合わせる

セレクターを空白を入れずに繋げることで、複数のセレクターを組み合わせることができます。

<WebExample>
  <style>
    {`
      .red {
        color: red;
      }
      .blue {
        color: blue;
      }
      .red.blue {
        color: purple;
      }
    `}
  </style>
  <p class="red">Hello</p>
  <p class="blue">World</p>
  <p class="red blue">Hello, World!</p>
</WebExample>

```css
.red {
  color: red;
}

.blue {
  color: blue;
}

.red.blue {
  color: purple;
}
```

```html
<p class="red">Hello</p>
<p class="blue">World</p>
<p class="red blue">Hello, World!</p>
```
