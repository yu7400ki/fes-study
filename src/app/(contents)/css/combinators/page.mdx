# 結合子

結合子は、セレクタ間の関係を指定するための記法です。
以下の構文で結合子を指定します。

```css
selector1 operator selector2 {
  property: value;
}
```

このとき、スタイルは`selector1`と`selector2`の関係に基づいて、`selector2`に適用されます。

## 子孫結合子

子孫結合子はスペース(`" "`)で表され、ある要素の子孫要素を指定します。

<WebExample>
  <style>
    {`
      main p {
        color: red;
      }
    `}
  </style>
  <main>
    <p>Hello</p>
    <div>
      <p>World</p>
    </div>
  </main>
</WebExample>

```css
main p {
  color: red;
}
```

```html
<main>
  <p>Hello</p>
  <div>
    <p>World</p>
  </div>
</main>
```

## 子結合子

直接子結合子は`>`で表され、ある要素の直接の子要素を指定します。

<WebExample>
  <style>
    {`
      main > p {
        color: red;
      }
    `}
  </style>
  <main>
    <p>Hello</p>
    <div>
      <p>World</p>
    </div>
  </main>
</WebExample>

```css
main > p {
  color: red;
}
```

```html
<main>
  <p>Hello</p>
  <div>
    <p>World</p>
  </div>
</main>
```

## 次兄弟結合子

次兄弟結合子は、`+`で表され、ある要素の直後に続く兄弟要素を指定します。

<WebExample>
  <style>
    {`
      p + p {
        color: red;
      }
    `}
  </style>
  <p>First</p>
  <p>Second</p>
  <p>Third</p>
</WebExample>

```css
p + p {
  color: red;
}
```

```html
<p>First</p>
<p>Second</p>
<p>Third</p>
```

### フクロウセレクタ

次兄弟結合子のよくある使われ方として、`* + *`があります。
これがフクロウの顔のように見えることから、通称フクロウセレクタと呼ばれます。

フクロウセレクタは、最初の要素以外の兄弟要素を指定します。
一般的に、積み重ねられた要素に垂直マージンを適用する際に使用されます。

<WebExample>
  <style>
    {`
      * {
        margin: 0;
        padding: 0;
      }
      article > * + * {
        margin-top: 1em;
      }
    `}
  </style>
  <article>
    <h1>Heading 1</h1>
    <p>Lorem ipsum dolor sit amet tempor hendrerit.</p>
    <p>
      Lorem labore et veniam kasd sed justo sanctus cum iriure lorem elitr dolor
      at. Aliquyam eirmod magna.
    </p>
    <h2>Heading 2</h2>
    <p>Gubergren justo et ipsum labore est lorem et dolore et.</p>
  </article>
</WebExample>

```css
article > * + * {
  margin-top: 1em;
}
```

```html
<article>
  <h1>Heading 1</h1>
  <p>Lorem ipsum dolor sit amet tempor hendrerit.</p>
  <p>
    Lorem labore et veniam kasd sed justo sanctus cum iriure lorem elitr dolor
    at. Aliquyam eirmod magna.
  </p>
  <h2>Heading 2</h2>
  <p>Gubergren justo et ipsum labore est lorem et dolore et.</p>
</article>
```

この例は、本書で使用されているスタイルの一部を抜粋したです。
段落や見出しの間にマージンが適用されていることがわかります。

## 後続兄弟結合子

後続兄弟結合子は、`~`で表され、ある要素の後続する兄弟要素を指定します。

<WebExample>
  <style>
    {`
      p ~ p {
        color: red;
      }
    `}
  </style>
  <p>First</p>
  <p>Second</p>
  <p>Third</p>
</WebExample>

```css
p ~ p {
  color: red;
}
```

```html
<p>First</p>
<p>Second</p>
<p>Third</p>
```
